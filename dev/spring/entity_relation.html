定义关系

关系可通过@ManyToOne、@OneToMany和@ManyToMany进行定义。

@ManyToOne

class : leap.orm.annotation.ManyToOne

用于定义多对一关系。

@OneToMany

class : leap.orm.annotation.OneToMany

用于定义一对多关系。

@ManyToMany

class : leap.orm.annotation.ManyToMany

用于定义多对多关系。
