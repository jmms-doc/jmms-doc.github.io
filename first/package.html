打包运行

开发完成后，我们需要把微服务打包部署到另外的环境，jmms支持两种打包方式。

1. Java Jar

这是Spring Boot提供的打包方式，把代码和配置打包成一个jar，通过java -jar xxx.jar执行。

在命令行执行：

$ jmms package

打包成功后，会在./target/目录中产生一个users-1.0.0-SNAPSHOT.jar文件，执行命令：

$ cd ./target
$ java -jar users-1.0.0-SNAPSHOT.jar

控制台将会打印启动信息。

2. Docker Image

需要准备好Docker运行环境

这是Docker提供的打包方式，先把代码打成Jar，然后通过Dockerfile构成镜像。

在命令行执行：

$ jmms docker build

构建成功后，执行命令：

$ jmms docker run

jmms对docker的build和run命令做了一个简单的包装，您也可以直接使用docker本身的命令进行操作
