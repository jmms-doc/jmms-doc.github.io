oauth2

Added in: v0.2.0

oauth2模块在接入OAuth2认证时才有效，用于获取或调用oauth2相关信息或接口。

Example:

获取当前请求的Access Token：

    const oauth2 = require('oauth2');

    const at = oauth2.accessToken;

oauth2.eixsts

Added in: v0.2.0

  • {boolean}

如果当前请求是有效的OAuth2请求返回true，否则返回false。

oauth2.accessToken

Added in: v0.2.0

  • {string}

获取当前OAuth请求的的Access Token。
