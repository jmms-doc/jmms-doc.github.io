环境变量

  • 简单类型的环境变量直接使用，如now
  • 复杂类型的环境变量可以使用name.property的方式访问，如user.id
Name Type Desc
now datetime 当前时间，精确到毫秒
currentTimeMillis long 当前时间的毫秒数表示
uuid string UUID
short_uuid string 对UUID进行编码的结果，长度更短，请看shortuuid
short_id string 请看shortid
user object 当前请求用户，具体属性请看$user
client object 当前请求客户端，具体属性请看$client
